### Explanation

beberapa penjelasan singkat setiap kegunaan folder nya buat klean klean yang masih cupu

#### Folder env

folder untuk klean klean yang mau pakai env banyak bisa di taro di folder ini.

cara config env nya baca <Link href="https://nextjs.org/docs" className="text-blue-500">di sini</Link>

#### Folder public

untuk menyimpan semua assets berupa image, svg, font, dan yang lainnya

#### Folder ioc

ini adalah bagian dari depedency injection dengan inversify.

detail untuk depedency injection bisa <Link href="https://nextjs.org/docs" className="text-blue-500">lihat di sini</Link>

#### Folder modules

folder modules berisi module - module atau bagian bagian dari fitur atau kerangka aplikasi yang berjalan secara terpisah.

detail untuk konsep modular cek <Link href="https://nextjs.org/docs" className="text-blue-500">di sini</Link>

#### Folder shared

di peruntukan untuk meyimpan file file yang bersifat global atau bisa digunakan ke module module lain
